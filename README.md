## Installation

Before the start, need to update Email and Password [here](https://github.com/unexpectedd0wn/abtasty/blob/main/cypress/fixtures/example.json), for the Successful authorisation case 

 1. Install
```sh
npm install
```
 2. Run test

```sh
npx cypress run 
```
